# Apps

Platform-specific PhonoDeck apps live here as the repository grows.

- `apple/` — Swift and SwiftUI apps for macOS, iOS, watchOS, and tvOS.
- `windows/` — native .NET / WinUI app.
- `web/` — React / TypeScript app.

The current macOS implementation still builds from `Sources/PhonoDeck`. Move code into this tree only through small, tested refactor phases.
